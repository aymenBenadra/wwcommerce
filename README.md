# WWCommerce

Another Wordpress project for the second brief of the WordPress series, This time we will be using the WooCommerce plugin to create a store, and then we will be adding some products to the store.

## Plugins used

- [WooCommerce](www.woocommerce.com)
- [Elementor](www.elementor.com)
- [GDPR Cookie Consent](www.wp-privacy.com)
- [WPForms Lite](www.wpforms.com)
- [TranslatePress](www.translatepress.com)
- [Yoast SEO](www.yoast.com)
